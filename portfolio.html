Hotel Pomelia

My very first website, created as an exercise during my Start2Impact course. A simple yet meaningful project, built entirely with HTML and CSS without any frameworks or external libraries. Since it was my first experience with coding, it was both exciting and frustrating—exciting when things worked, frustrating when they didn't!

The result is a static, non-responsive, single-page website, but it marks the beginning of my journey. Looking back, it's incredible to see how much I've improved since then!

My Portfolio

The website you're currently browsing is my first complex project. Unlike my first website, this one features multiple pages, smooth transitions, and a fully responsive design. I used HTML, CSS, Sass, and Bootstrap, which helped me better structure my code and build a more polished website. It's also my first project to be hosted on GitHub.

I'm particularly proud of the project cards, which adapt seamlessly to different screen sizes, and the color palette, which I feel represents my style and values well. I chose to organize content across multiple pages rather than creating a single long-scrolling page, making navigation cleaner and more user-friendly.

In the future, I plan to add a dark mode toggle, scroll effects, custom illustrations, and transform the project section into an interactive slideshow.
